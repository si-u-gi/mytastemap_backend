MySQL을 Docker 컨테이너로 설치 (데이터 영구)
MySQL 데이터용 Volume (최초 1회)
docker volume create mytastemap-mysql-data

MySQL 컨테이너 실행 (최초 1회)
docker run -d \
  --name mytastemap-mysql \
  -e MYSQL_ROOT_PASSWORD=1234 \
  -e MYSQL_DATABASE=mytastemap \
  -p 3306:3306 \
  -v mytastemap-mysql-data:/var/lib/mysql \
  mysql:8.0

접속
docker exec -it mytastemap-mysql mysql -u root -p
